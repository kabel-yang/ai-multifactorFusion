# ai-multifactorFusion
 基本面 + 技术面 + 舆情多因子融合


代码结构说明

核心模块

DataFetcher​ - 统一数据接口，支持模拟/真实数据切换

AdvancedSentimentAnalyzer​ - 基于FinBERT的专业金融情感分析

AdvancedFactorEngine​ - 技术+基本面+舆情多因子计算

FeatureEngineer​ - 特征工程（行业中性化、标准化等）

AdvancedStockSelector​ - LightGBM + 贝叶斯优化模型

AdvancedBacktester​ - 完整回测框架


模块流程：模拟数据生成 → 技术因子计算 → 舆情情感分析 → 因子合并 → 模型训练 → 信号生成 → 回测


核心功能：

真实语义新闻生成​ - 模拟带真实语义的新闻文本

FinBERT金融情感分析​ - 专业的预训练金融NLP模型

行业中性化处理​ - 避免行业暴露风险

改进模型：使用LightGBM + 贝叶斯优化调参

高级可视化​ - 收益曲线、热图等多维度分析

因子计算：动量、波动率、成交量 + 舆情因子

标签构建：未来5日收益率，前30%为买入信号

风控要点：

严格避免未来函数（使用 .shift()滞后处理）

滚动训练防止过拟合

行业中性控制（示例中简化）

完整回测框架还需要加上包含交易成本、滑点、仓位管理、风控处理

可以增加特征：添加基本面、资金流、龙虎榜等因子

这个示例展示了完整的AI量化策略开发流程。实际应用中，每个模块都需要进一步优化和严格测试。
